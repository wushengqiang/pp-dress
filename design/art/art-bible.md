# Art Bible: 每日穿搭 (Dress Up Daily)

*Created: 2026-06-05*
*Status: In Progress*

---

## 1. Visual Identity Statement

### 一线视觉准则

> 像清晨的第一缕暖光——不耀眼，不催促，只是安静地照亮你的选择。

当任何视觉决策出现歧义——颜色、间距、动画、UI密度——统一问一句：*这是清晨暖光，还是正午刺阳？它在耐心陪伴，还是在催促玩家？它在照亮玩家的选择，还是在评判它？*

### 支撑原则

**一、暖光如抱**（服务「每日陪伴」）

整个视觉空间被暖粉色光包裹——无锐利阴影、无冰冷高光，一切都在轻声说：这里很安全。

- *设计试金石*：不确定颜色或光影时，问自己"这是拥抱还是刺眼？"选拥抱。

**二、笔触回声**（服务「即时有感」）

每一次拖拽、放下、切换都有手绘质感的即时反馈——粒子如笔触飞溅，高光如水彩晕开。让玩家感觉这幅画在回应她，而非机器在运转。

- *设计试金石*：设计交互反馈时，问自己"这像画在回应玩家，还是机器在运转？"选前者。

**三、衣为焦点**（服务「随心搭配」）

界面后退，穿搭居中。用留白、浅底色和克制的 UI 确保玩家的搭配始终是画面上最重要的事物——穿搭是主角，UI 是安静的舞台。

- *设计试金石*：不确定 UI 元素是否必需时，问自己"它在和穿搭抢注意力吗？"如果是，去掉。

### 原则与支柱映射

| 游戏支柱 | 视觉原则 | 运作方式 |
|---------|---------|---------|
| 每日陪伴 | 暖光如抱 | 暖色调 + 柔和光影创造情感安全感 |
| 即时有感 | 笔触回声 | 手绘质感的反馈粒子让每次交互都有生命 |
| 随心搭配 | 衣为焦点 | 留白 + 克制 UI = 玩家的穿搭永远占据视觉中心 |

### 美学交集

三条原则均落在 **日式小清新**（柔和色调、干净构图、呼吸感、自然光意象）与 **温暖手绘风**（圆润线条、手绘质感、无锐利边角）的交集内。两种美学互为增强，无冲突。

## 2. Mood & Atmosphere

### 主菜单/首页
- **核心情绪**: 被温柔迎接的安心感
- **光线特征**: 清晨暖阳 (~3500K)，柔光，低反差
- **氛围关键词**: 安静 · 温暖 · 期待 · 柔软
- **能量层级**: calm
- **情绪载体**: 阳光透过薄窗帘洒在桌面的光斑，微微晃动

### 衣橱换装
- **核心情绪**: 专注于自我表达的愉悦
- **光线特征**: 明亮的上午光，清晰不刺眼，中反差
- **氛围关键词**: 明亮 · 自由 · 雀跃 · 沉浸
- **能量层级**: measured
- **情绪载体**: 拖拽时服装部件跟随指尖的微动画，放下时有柔光涟漪扩散

### 每日场景
- **核心情绪**: 温馨的日常陪伴感
- **光线特征**: 午后/黄昏暖光（随场景微调），中低反差
- **氛围关键词**: 温暖 · 亲密 · 真实 · 日常
- **能量层级**: contemplative
- **情绪载体**: 角色身着玩家搭配的服装，侧身微笑的姿态

### 晚安收尾
- **核心情绪**: 温柔的收束与满足
- **光线特征**: 夜晚台灯暖光 (~2700K)，低反差，柔暗
- **氛围关键词**: 安宁 · 满足 · 私密 · 柔软
- **能量层级**: calm
- **情绪载体**: 一盏台灯的暖色光晕包裹着身着睡衣的角色

### 解锁新衣
- **核心情绪**: 小小惊喜的雀跃
- **光线特征**: 短暂提亮 + 柔光粒子，稍高于常态的饱和度
- **氛围关键词**: 惊喜 · 珍贵 · 期待 · 珍藏
- **能量层级**: measured（不过度——不与"安静陪伴"的主基调冲突）
- **情绪载体**: 新服装从柔光粒子中浮现，光晕一圈圈扩散后轻轻收拢

### 设计一致性

五个状态统一在"暖光"基调下，通过色温 (~2700K–~5000K) 和反差变化区分情绪。能量最高点（解锁新衣）仍控制在 measured，不进入 frenetic——符合"不催促"的核心承诺。每个状态的情绪载体均为"光"或"服装微动画"，对应暖光如抱 + 衣为焦点。

## 3. Shape Language

### 角色剪影 —— 安静的画布
基础剪影以柔和竖弧形为骨架：微圆肩线、自然垂臂、约 5.5 头身比例。轮廓刻意克制，不设夸张形——角色是画布，服装才是笔触。缩略图下清晰可辨，部件换装一目了然。**→ 衣为焦点**

### 场景几何 —— 拥抱的曲线
场景以圆润有机形为主导：弧线窗框、圆角家具、波浪收边的装饰线。锐利直角被系统性回避——硬角属于冰冷的城市，曲线才属于这个温暖的小世界。**→ 暖光如抱**

### UI 形状 —— 安静的舞台
UI 继承世界的圆润基因，但更轻更透：10-14px 圆角胶囊按钮、16px 圆角面板、衣橱网格为圆角浅影卡片。底色低不透明度，描边纤细——UI 是安静的舞台，从不出演。**→ 衣为焦点**

### 视觉层次 —— 穿搭为王
层次秩序：穿搭 > 角色 > 场景 > UI。穿搭居画面中心，享有最大面积、最高对比度与暖光聚集。背景以暖灰低对比柔化退后，UI 以半透明浮层栖于边缘。**→ 随心搭配**

### 服装廓形 —— 风格的灵魂
三种廓形传达不同情绪，均在日式小清新范围内——不夸张、不戏剧化：
- **温柔系**: A 字弧线 —— 柔软放松，如连衣裙、荷叶边
- **利落系**: 微收腰直线 —— 自信明快，如衬衫、西装裤
- **舒适系**: 松弛方廓 —— 自在日常，如卫衣、阔腿裤

换装时剪影切换配合手绘微动画，即「即时有感」的核心时刻。**→ 笔触回声 + 衣为焦点**

## 4. Color System

### 核心策略
环境/UI **后退**，衣物 **前进**。整个色彩体系服务于"衣为焦点"——环境色必须克制，让服装成为画面上唯一的主角。

### 主色调（7色）

| 色名 | 色值 | 角色与含义 |
|------|------|-----------|
| **晨光白** | `#FDF6EE` | 画布。清晨第一缕光落在纸上的颜色。占画面 60% 以上，是所有"呼吸感"的来源。 |
| **暖杏** | `#F0CFBA` | 温度。阳光透过薄纱窗帘。用于高亮区域、家居场景主调、UI 选中态。 |
| **樱茶** | `#E4B8A8` | 柔软。樱花与焙茶的中间色——暖但不甜腻。用于 UI 按钮底色、温柔提示。 |
| **若草** | `#C6D6B8` | 安静。日本传统"若草色"——初春草芽。代表户外/自然场景的底色基调。 |
| **水浅葱** | `#B5CCD2` | 清澈。带暖调的浅蓝，是晨空而非冷海。用于平静状态、过渡场景。 |
| **山吹茶** | `#C5A88A` | 接地。温润的赭褐色，手绘线稿的"墨色"。次要文字、分割线、图标。 |
| **小豆色** | `#8B6F6F` | 重量。暖调最深色——即便暗色也保留温度。正文、关键图标、焦点态。 |

### 语义色用法

| 语义 | 配色 | 场景 |
|------|------|------|
| 暖意 / 归属感 | 暖杏 + 晨光白 | 家的场景、确认反馈 |
| 平静 / 自然 | 若草 + 水浅葱 | 户外、等待状态 |
| 聚焦 / 高亮 | 留白 + 微弱樱茶辉光 | 衣橱换装界面 |
| 文字层级 | 山吹茶 → 小豆色 → 暖杏 | 次要 → 正文 → 装饰 |

### 场景背景色温

| 场景 | 基调色 | 色温 | 理由 |
|------|--------|------|------|
| 卧室 | 晨光白 + 暖杏晕染 | ~3500K | 安全空间，晨光包裹 |
| 街道 | 若草 + 少量水浅葱 | 中性偏暖 | 过渡空间，日常风景 |
| 咖啡馆 | 暖杏为主 + 樱茶点缀 | ~3200K | 社交空间，亲密包裹 |

### UI 调色板偏离规则

UI 比世界层多半级对比度以保证可读性，但不跳脱整体氛围：
- **按钮填充**: 樱茶 `#E4B8A8`，悬停加深 10% 亮度
- **选中/激活态**: 暖杏 `#F0CFBA` 外发光，不加描边
- **禁用态**: 山吹茶 `#C5A88A` 40% 透明度
- **危险/删除**: 小豆色 `#8B6F6F`（不引入红色——破坏暖光调性）

### 色觉无障碍

**高危色对**（需形状/图标兜底）：

| 色对 | 风险类型 | 兜底措施 |
|------|----------|----------|
| 若草 vs 水浅葱 | 红绿色盲难区分 | 场景切换需**图标 + 标签文字** |
| 暖杏 vs 樱茶 | 亮度接近 | 选中态加**外发光 + 缩放**，禁用态加**纹理叠加** |
| 暖杏 vs 山吹茶 | 色相接近 | 重要操作按钮加**圆角形状区分** |

所有需要区分状态的地方，必须有"颜色 + 形状/图标/位置"至少两个信号源。

## 5. Character Design Direction

### 主角色视觉原型
16-18 岁少女，5.5 头身，邻家气质。默认面容素净，黑色中长发，白 T 恤 + 淡蓝短裤——朴素如未完成的素描，邀你填色。站姿略侧身、重心偏右、双手自然交叠于身前、头微偏。气质属"癒し系"：温柔安静，让人想照顾，而非仰望。**她是画布，不是画作。**

### 表情与姿态
表情收窄至 4 种：平静、微笑、害羞、小惊喜。通过眉眼微弧与嘴角轻抬传达，不做夸张符号（汗滴、青筋），不引入大哭/愤怒——极端情绪破坏陪伴安全感。默认姿态为自然放松的日常站姿。

### 部件分层规则（Z 轴底→顶）

| 层 | 类别 | 说明 |
|----|------|------|
| 1 | 妆容 | 直接映射面部底图，无透明边 |
| 2 | 下装 | 覆盖腿部 |
| 3 | 鞋子 | 覆盖脚部，与下装下缘衔接 |
| 4 | 上衣 | 覆盖躯干，下缘与下装腰线锚点对齐 |
| 5 | 配饰 | 覆盖上衣，需透明边缘（围巾、项链） |
| 6 | 发型 | 最顶层，纯顶层简化（MVP）。发型覆盖衣领，短发发尾与颈锚点对齐 |

### 设计约束（不可变项）
锁定项（所有服装部件的适配基准）：
- 头身比 5.5:1
- 四个关节锚点：颈根、肩峰、腰线、踝线
- 身体轮廓：肩弧、臂长、颈长、躯干长
- 皮肤色：暖杏 `#F0CFBA`（禁止部件修改皮肤区域）
- 五官位置与比例（妆容只能叠色，不可移形）

可变项：所有服装、发型、妆容、配饰。

### 角色作为画布
核心视觉张力：角色必须**足够可爱让人在意，又足够中性让服装永远成为焦点**。
- 素颜状态是"等待被完成"的线稿——略带未完成感，激发玩家完成欲
- 默认白 T + 短裤刻意朴素，传递"她需要你的品味"
- 五官克制：柔和眉眼、无猫眼/厚唇/特殊眉形——风格留给妆容定义

## 6. Environment Design Language

### 场景设计哲学
背景是情绪舞台，非探索空间。场景唯一的任务是：用光线与色调为穿搭提供安静画布，不与之争抢视线。玩家进入的反应应是"好舒服"，而非"有什么可点的"。

### 建筑与空间风格
日系现代休闲 × 温馨家居。所有几何均为圆角曲线，拒绝直角硬边。

- **卧室**: 晨光薄纱帘、木地板、圆角矮床、散落靠垫。最安全、最私密的空间。
- **街道**: 柔和街景剪影、行道树、面包店橱窗暖光。过渡空间，不经意的日常风景。
- **咖啡馆**: 圆木桌、热咖啡、窗边座、书架一角。社交空间，更亲密、更包裹。

### 纹理哲学
水彩薄涂 + 纸纹基底。边缘微渗、色有细微不匀，保留笔触痕迹。非数码平滑——温暖来自不完美。"刚画完的水彩速写"。

### 道具密度
取日式留白与生活感的平衡点。每场景 5-8 件叙事道具，不超过 10 件。每件道具都有叙事理由，无纯装饰。

### 环境叙事指南
不说一字讲故事。床头半杯水 + 翻开的书 = 昨夜有人在此。窗台多肉 + 洒落阳光 = 被精心对待的空间。咖啡馆桌上两只杯子 = 她刚才不是一个人。所有细节指向同一件事：她过着安静、温暖、被珍惜的日常。

### 场景切换过渡
0.8 秒晨光白 (`#FDF6EE`) 淡入淡出。如翻过一页绘本，而非加载下一关。伴极轻"纸页翻动"质感音效。视觉上不打断、不抖动。

## 7. UI/HUD Visual Direction

### UI 设计哲学
界面是舞台，穿搭是主角。UI 只服务穿搭，不争目光。目标：玩家眼中唯有角色与服装，界面如晨光透帘——在而不显。**→ 衣为焦点**

### 布局架构
核心空间语法：穿搭居中，UI 浮边。

- **衣橱界面**: 左侧竖列类别胶囊标签（视觉 14px，触控热区 44×44px），底部横滚服装网格（16px 圆角卡片，间距 ≥8px），角色居画面中央。卡片不透明度 ≥85%。
- **场景对话**: 底部半透明浮层，不越中线。
- **菜单/晚安**: 居中极简，大面积晨光白留白。
- **铁律**: 任何面板不得覆盖角色主体 30% 以上。
- **响应式**: ≥768px 左右分栏，<768px 上下分区。拖拽坐标统一使用画布相对坐标。窄屏拖拽触发 `preventDefault` 阻止页面滚动。

### 字体
圆体无衬线，温润如手写便签，不机械不幼稚。

| 层级 | 字重 | 大小 | 用途 |
|------|------|------|------|
| 标题 | Light | 24px | 场景标题、章节名 |
| 正文 | Regular | 14px | 对话文本、描述 |
| 标签 | Medium | 14px | 类别名、按钮 |
| 辅助 | Regular | 12px | 提示、标注 |

正文色使用加深版小豆色 `#7A5252`，在晨光白上对比度 ≥4.6:1（WCAG AA）。

### 图标
线描勾勒风，与手绘美学同源。1.5-2px 描边，略带手绘微不规则感——拒绝完美矢量机械味。尺寸 16-20px。默认色山吹茶 `#C5A88A`，激活态小豆色 `#8B6F6F`。功能定位：辅助识别，不与服装争视觉权重。

### UI 动画
柔和 ease-out + 微回弹，如画笔轻落纸面。微交互 200-300ms，面板滑入 400-500ms。循「笔触回声」原则：每帧动画带轻微延迟与不规则——像有人在为你翻绘本的下一页，而非机器在切屏。

### 交互反馈

| 动作 | 视觉效果 | 时长 |
|------|---------|------|
| **拖起** | 部件微提（1.05x 缩放 + 4px 暖调柔阴影） | 即时 |
| **悬停有效区** | 角色轮廓泛起暖杏 `#F0CFBA` 柔光外晕 + 4px 脚底暖光投影锚点 | 即时 |
| **放下成功** | 水彩晕开涟漪 + 手绘笔触粒子飞溅 | ~400ms |
| **触按** | 0.95x 轻缩 + 樱茶底色微亮，松手回弹 | ~200ms |
| **转场** | 0.8 秒晨光白 `#FDF6EE` 翻页式淡入淡出 | 800ms |
| **拖拽源槽位** | 保留虚线轮廓（ghost target，不透明度 40%） | 持续至放下 |

### 无障碍

- **替代操作**: 并行支持"点击选中 → 点击角色部位穿上"，确保运动障碍玩家可操作
- **触控热区**: 所有可交互元素触控热区 ≥44×44px
- **色觉安全**: 所有状态区分必须有"颜色 + 形状/位置"双信号源
- **文字对比**: 正文 ≥4.5:1（WCAG AA）

## 8. Asset Standards

### 文件格式与分辨率

| 资产 | 格式 | 画布尺寸 |
|------|------|---------|
| 角色底图 + 服装部件 | PNG-24 RGBA | **1024×1536**（统一画布，锚点对齐） |
| 场景背景 | PNG-24 | **2048×1536**（4:3 横幅） |
| UI 元素 | PNG-24 | 原生 48×48px |

所有服装部件使用与角色底图**完全相同的画布尺寸和原点**，四个关键锚点（颈根、肩峰、腰线、踝线）坐标固定不变——这是部件像素对齐的前提。

### 命名规范

`[类别]_[名称]_[变体].[ext]`，前缀即类别，自文档化且排序友好：

| 类别 | 前缀 | 示例 |
|------|------|------|

### 导入设置（Godot 4.6）

| 参数 | 设置 | 理由 |
|------|------|------|
| 压缩 | Lossless | 透明边缘不容有任何压缩伪影 |
| 过滤 | Linear | 水彩柔和质感，非像素风 |
| Mipmaps | On | Web 视口缩放时防锯齿 |
| VRAM 压缩 | 关闭 | WebGL 2.0 ETC2 对手绘纹理损失明显 |

### Web 加载策略（三层渐进）

1. **首帧关键资产**（同步加载，~5MB）：角色素体 + 上衣/下装/鞋子默认款 + UI 框架 + 当前场景背景。首帧 < 2s。
2. **当前类别预加载**（异步）：玩家切换类别时后台加载该类其余服装。使用 `ResourceLoader.load_threaded_request()`。
3. **其他类别延迟加载**：仅在该类别首次被打开时加载。

### 性能约束

- 满屏 Sprite2D 节点：~15-20 层（远低于 50 draw call 上限）
- 总纹理内存：≤80MB（256MB Web 总预算内）
- Godot 端使用 `Sprite2D.offset` 微调（≤2px），其余靠画布级别对齐

### 源文件格式

推荐 **Krita (.kra)**——免费开源，水彩笔刷引擎在开源工具中最出色，原生支持图层分组与导出，与"水彩薄涂 + 纸纹"的纹理哲学完美匹配。备选：Procreate（iPad）、Photoshop（已有授权时）。

## 9. Reference Direction

| # | 参考源 | 学什么 | 避开什么 |
|---|--------|--------|---------|
| 1 | **Nintendo 3DS「Girls Mode」** | 衣物独立留白的陈列节奏；"试穿即预览"的即时反馈模式——每件单品被当作值得端详的对象，而非列表项 | 3D 模型的塑料光泽感；商场橱窗式的消费主义语调 |
| 2 | **岩田圭子 水彩人物** | 水彩在轮廓边缘的自然晕开——颜色在交界处互渗，产生"刚画完还湿润"的鲜活感 | 偏冷、偏灰的实验性色调；大幅留白的构图方式（本作需以服装填充画面重心） |
| 3 | **吉卜力背景美术（男鹿和雄系）** | 阳光不是"黄色滤镜"，而是物体受光面的色温偏移——暖杏色高光，阴影偏紫灰而非纯黑。光在桌布、头发、衣物褶皱上"停留"的方式 | 奇幻尺度的场景纵深；极高细节的自然背景（本作背景应退后，让衣服成为焦点） |
| 4 | **ほぼ日手帳 每日页面** | 日期作为视觉锚点的排版逻辑；手写文字作为肌理而非信息；纸纹底色的克制运用——页面"有记录痕迹但不拥挤"的分寸 | 真实手帐的信息密度和功能性排版；商业文具的过度设计感 |

四个参考分别指向：**展示节奏、笔触质感、光的温度、日记氛围**——互不重叠，彼此补足。
